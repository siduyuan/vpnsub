# VPN Subscription

自动更新的 VPN 订阅文件

## 订阅链接

### 推荐（jsDelivr CDN，国内访问优化）
```
https://cdn.jsdelivr.net/gh/siduyuan/vpnsub@main/sub.txt
```

### 备选（GitHub Raw）
```
https://raw.githubusercontent.com/siduyuan/vpnsub/main/sub.txt
```

### GitHub Pages（需要手动启用）
```
https://siduyuan.github.io/vpnsub/sub.txt
```

## 更新频率

每 6 小时自动更新一次

## 说明

- 订阅文件由服务器自动筛选并推送
- 所有节点都经过 TCP 连接测试
- 建议在客户端中进行真连接测试以选择最佳节点
- 当前节点数：74 个

## 使用方式

在 V2RayN/V2RayNG 等客户端中：
1. 添加订阅链接
2. 更新订阅
3. 进行真连接测试
4. 选择延迟最低的节点使用
